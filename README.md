# Portafolio 

El propósito de esta página web es mostrar el portafolio de mis proyectos y mi experiencia. Incluye una descripción breve de mi motivación, experiencia, proyectos, artículos publicados y formas de contacto. 

También incluye imágenes alternativas en la carpeta `imagenes` en caso de que se desee personalizar la imagen principal del desarrollador, además de animación a la escritura de la página web con JavaScript.
